# Documentation Home

This documentation is organized into modular areas:

- [`phases/`](https://github.com/haseebraza715/failure-aware-ocr-rag/blob/main/docs/phases/index.md): phase-scoped planning, methodology, checklists, and completion records
- [`repo_handbook/`](https://github.com/haseebraza715/failure-aware-ocr-rag/blob/main/docs/repo_handbook/index.md): cross-phase repository knowledge and stable architecture
- [`reports/`](https://github.com/haseebraza715/failure-aware-ocr-rag/blob/main/docs/reports/index.md): phase summary reports for supervisors and reviewers
- [`archives/`](https://github.com/haseebraza715/failure-aware-ocr-rag/blob/main/docs/archives/index.md): documentation-facing indexes of logs and artifacts

## Update Policy

- Update phase docs inside `docs/phases/phaseX/` while a phase is active.
- After a phase is completed, update:
  - `docs/reports/phaseX_report.md`
  - `docs/archives/phaseX_archive_index.md`
  - relevant sections in `docs/repo_handbook/`
