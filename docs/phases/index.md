# Phases Documentation

Phase-scoped documentation is organized as:

- [`phase0/`](./phase0/index.md)
- [`phase1/`](./phase1/index.md)
- [`phase2/`](./phase2/index.md)
- [`phase3/`](./phase3/index.md)

## Conventions

- Each phase folder contains methodology, acceptance, and completion evidence.
- Cross-phase summaries live in `docs/reports/`.
- Whole-repo non-phase knowledge lives in `docs/repo_handbook/`.
